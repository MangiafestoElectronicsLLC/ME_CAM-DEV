## 1) Daily Use
1. Open the camera web page on your phone, tablet, or PC.
2. Sign in with the customer username and password provided.
3. Live view appears on the dashboard.
4. Use Nanny Cam Mode toggle:
- ON: live view only, no motion recording.
- OFF: motion recording and alerts run normally.

## 2) Talk Through The Camera
1. In the dashboard, use the Speak controls near Nanny Cam Mode.
2. Option A: Type text and press Speak Text.
3. Option B: Press Start Voice, speak into your phone/PC microphone, then press Send Voice.
4. Sound will play through the camera device speaker.

## 3) Motion Clips
1. Open Events or Clips from the dashboard top buttons.
2. View recent motion events.
3. Play and download motion clips if needed.

## 4) Text Alerts
1. Open Config.
2. Enable Send text alerts for motion.
3. Enter Text Alert Number with country code (example: +1234567890).
4. Save Configuration.

## 5) WiFi Update
1. Open Config.
2. Update WiFi SSID and password.
3. Save Configuration.
4. Reboot if instructed.

## 6) Troubleshooting
- If live view is black/test mode: reboot device and verify camera cable.
- If audio speak fails: check speaker/USB audio is connected.
- If alerts do not send: confirm phone number and provider setup in Config.
