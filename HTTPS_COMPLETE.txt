API ENDPOINTS (All HTTPS-compatible)
================================================================================

Motion Events:
  GET  /api/motion/events?hours=24&limit=50
       Returns: List of motion events with timestamps

Stream Control:
  GET  /api/stream
       Returns: MJPEG video stream (works with HTTPS)
  POST /api/stream/quality
       Body: {"quality": "low|standard|high|ultra"}

Storage:
  GET  /api/storage
       Returns: GB used, free space, retention settings
  POST /api/clear-storage
       Clears old recordings

Recordings:
  GET  /api/recordings
       Returns: List of video files
  GET  /api/download/{filename}
       Downloads a recording file
  POST /api/delete/{filename}
       Deletes a recording

System:
  GET  /api/status
       Returns: System status, uptime, battery
  GET  /api/camera/stats
       Returns: Streaming mode, FPS, resolution

All endpoints automatically work with:
  - HTTP (if HTTPS certs not present)
  - HTTPS (if certs present)
  - Domain names (ME_CAM.com)
  - IP addresses (10.2.1.47)
